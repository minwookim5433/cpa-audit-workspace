저장됨
  1. 시험지 PDF 선택
  2. 오른쪽 답안지 작성
  3. 중간에 멈추려면 저장하고 나가기 — 다음에 같은 시험지를 열면 이어서 풀 수 있습니다
  4. 시험 종료 후 PDF 저장

풀이 내용(답안·주석·페이지·타이머)은 자동 저장됩니다. 북마크는 시험지 페이지 위치 표시용이며, 이어 풀기는 자동 저장으로 처리됩니다.
